# 1157 - Divisors I

Read an integer N and compute all its divisors.

**Input**<br>
The input file contains an integer value.

**Output**<br>
Write all positive divisors of N, one value per line.

| Input Sample | Output Sample          |
|:-------------|:-----------------------|
| 6            | 1 <br> 2 <br> 3 <br> 6 |
